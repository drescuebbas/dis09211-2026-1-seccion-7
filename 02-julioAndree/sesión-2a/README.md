# 16/03/26 Mapas Mentales  
Observación de usuario: Vista intencionada.  
## Técnicas:  
  * Mapa de Empatia: Herramienta visual.
  * Mapa de flujo: Mapa (ej: recorrido, acumulación de gente)
  * Entrevistas
  * Shadowing: Seguir a la persona.
  * Sondas: Registros desde la perspectiva de la persona.
    
**Nunca suponer y siempre empatizar con las experiencias humanas**  
### Entrevistas  
Técnica cualitativa, es una conversación guiada.  
No Encuesta - No Interrogatorio  
### Tipos:  
**Estructura**
* Semiestructurada: siempre siguiendo una pauta pero de conversación abierta.
 
**Contexto**
* Entrevista Corta/Profunda
* De Contexto.  
* De Expertos.    
* De Extremos.
* De Interseptos: Se busca info puntual.

**Indagar en motivaciones profundas** **-** **Empatizar realidades** **-** **Conocer experiencias**
